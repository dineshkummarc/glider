JavaScript content glider

Glider slides content horizontally or vertically inside a containing div. It is based on Prototype.js and script.aculo.us.

Example

As an example, simply checkout how the navigation on http://simplabs.com works.

Installation

Glider is available from github: http://github.com/simplabs/glider.
